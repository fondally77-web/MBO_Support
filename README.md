# 個人用MBOシステム

将来の拡張可能性を持った個人用MBO（目標管理）システムです。

## 特徴

- **React 18 + TypeScript 5** - モダンな開発環境
- **TailwindCSS** - 柔軟なスタイリング
- **ローカルストレージ** - データ永続化
- **等級システム対応** - 7段階資格等級 + 11段階役割等級

## 等級システム

### 資格等級（7段階）
| コード | 名称 | レベル | 説明 |
|--------|------|--------|------|
| TM1 | 一般職 | 1 | 定型反復業務処理 |
| TM2 | 上級一般職 | 2 | 複雑困難事項処理 |
| L1 | 初級専門職 | 3 | 実務処理能力・後進指導 |
| L2 | 中級専門職 | 4 | 企画判断力・専門知識技術活用 |
| L3 | 上級専門職 | 5 | 複雑困難業務遂行・実務統括 |
| M | 管理職 | 6 | 部署具体策企画立案・一般職指導育成 |
| SM | 経営職 | 7 | 経営的視野・重要施策企画立案 |

### 役割等級（11段階）
- **TM1系列**: TM1-1（初級）、TM1-2（中級）、TM1-3（上級）
- **TM2系列**: TM2-1（初級）、TM2-2（中級）、TM2-3（上級）
- **L1系列**: L1-1（初級）、L1-2（中級前半）、L1-3（中級後半）、L1-4（上級前半）、L1-5（上級後半）

## 拡張性

このシステムは以下の拡張に対応しています：

1. **等級システムの拡張**
   - `src/types/grade.ts` で新しい等級を追加
   - `src/data/gradeMaster.ts` でマスタデータを更新

2. **新機能の追加**
   - `src/components/` に新しいコンポーネントを追加
   - `src/types/` に必要な型定義を追加

3. **データストレージの変更**
   - `src/utils/localStorage.ts` を変更することで、バックエンドAPIへの切り替えも可能
