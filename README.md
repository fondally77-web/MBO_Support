# 個人用MBOシステム

将来の拡張可能性を持った個人用MBO（目標管理）システムです。

## 特徴

- **React 18 + TypeScript 5** - モダンな開発環境
- **TailwindCSS** - 柔軟なスタイリング
- **ローカルストレージ** - データ永続化
- **等級システム対応** - 7段階資格等級 + 11段階役割等級

## 等級システム

### 資格等級（7段階）
| コード | 名称 | レベル | 説明 |
|--------|------|--------|------|
| TM1 | 一般職 | 1 | 定型反復業務処理 |
| TM2 | 上級一般職 | 2 | 複雑困難事項処理 |
| L1 | 初級専門職 | 3 | 実務処理能力・後進指導 |
| L2 | 中級専門職 | 4 | 企画判断力・専門知識技術活用 |
| L3 | 上級専門職 | 5 | 複雑困難業務遂行・実務統括 |
| M | 管理職 | 6 | 部署具体策企画立案・一般職指導育成 |
| SM | 経営職 | 7 | 経営的視野・重要施策企画立案 |

### 役割等級（11段階）
- **TM1系列**: TM1-1（初級）、TM1-2（中級）、TM1-3（上級）
- **TM2系列**: TM2-1（初級）、TM2-2（中級）、TM2-3（上級）
- **L1系列**: L1-1（初級）、L1-2（中級前半）、L1-3（中級後半）、L1-4（上級前半）、L1-5（上級後半）

## 機能

### 現在実装されている機能

#### 基本機能
- ✅ **プロフィール設定**（名前、部署、等級情報、ポジション）
  - 等級別ガイダンス表示
  - リアルタイム入力検証
- ✅ **等級マスタデータ管理**
  - 7段階資格等級（レベル1-7）
  - 11段階役割等級
  - 詳細な役割説明
- ✅ **プロフィール表示**
  - モダンなカードデザイン
  - 等級情報の視覚的表示
- ✅ **ナビゲーションバー**
  - グラデーションデザイン
  - ユーザー情報表示
  - ページ遷移メニュー

#### AI機能
- ✅ **AI目標アドバイザー**（OpenAI API統合）
  - 等級レベルを考慮した目標分析
  - SMART基準の自動評価（5項目）
    - Specific（具体的）
    - Measurable（測定可能）
    - Achievable（達成可能）
    - Relevant（関連性）
    - Time-bound（期限）
  - 等級レベル適合性判定
  - 具体的な改善提案（3-5点）
  - 総合スコア表示（0-100点）

- ✅ **AI目標提案機能**（OpenAI API統合）
  - 仕事内容と等級レベルに基づいた目標提案
  - 3つのカテゴリ別提案（挑戦目標・業務目標・人財目標）
  - 1回の提案で3-5件の目標を自動生成
  - 難易度レベル表示（基礎・標準・挑戦的）
  - AI推薦理由の詳細説明
  - 提案から選択してフォームに自動反映
  - 提案履歴の保存・閲覧機能

- ✅ **挑戦目標 対話型アシスタント**（OpenAI API統合）
  - チャット形式で段階的に目標を詳細化
  - 探索的アプローチ（状況理解→提案→詳細化）
  - ユーザー主導の対話フロー
  - 3段階の達成基準設定（着手・実用・改善）
  - MBO文書フォーマットでの出力
  - 文字数調整やスケジュール詳細化に対応
  - メッセージのコピー機能

- ✅ **業務目標 対話型アシスタント**（OpenAI API統合）
  - 段階的な質問（最大9個）で不明点を解消
  - 3フェーズの情報収集プロセス
  - 組織目標との紐づけを重視
  - 定量的な成果指標の設定支援
  - 役割の明確化（他メンバーとの分担）
  - 達成度評価基準と月次プロセスの提案
  - フォーマット調整（表形式、箇条書き等）

#### データ管理
- ✅ **データインポート・エクスポート**
  - JSON形式でのバックアップ
  - データ復元機能
  - データ統計情報表示
- ✅ **設定画面**
  - OpenAI APIキー管理
  - データサイズ表示
  - 全データ削除機能
  - システム情報表示

#### その他
- ✅ **ローカルストレージ**でのデータ永続化
- ✅ **レスポンシブデザイン**（モバイル・タブレット・デスクトップ対応）
- ✅ **エラーハンドリング**と**ローディング表示**

## 使用方法

### 1. 初回セットアップ
1. アプリケーションを起動
2. プロフィール設定画面で基本情報を入力
   - 名前
   - 部署
   - 資格等級
   - 役割等級
   - ポジション
   - 等級取得日

### 2. OpenAI APIキーの設定（AI機能を使用する場合）
1. 設定画面を開く
2. [OpenAI API Keys](https://platform.openai.com/api-keys)でAPIキーを取得
3. APIキーを入力して保存
4. 検証が成功すれば、AI機能が使用可能になります

### 3. AI目標アドバイザーの使用
1. ナビゲーションメニューから「AI目標アドバイザー」を選択
2. 目標情報を入力
   - 区分（業務遂行、業務改善、人材育成、スキル習得、その他）
   - 目標タイトル
   - 目標の説明
   - 達成基準（100%）
   - 達成基準（110%）※任意
3. 「AI分析を実行」ボタンをクリック
4. 分析結果を確認
   - 総合スコア
   - SMART基準評価
   - 等級レベル適合性
   - 改善提案

### 4. AI目標提案機能の使用

#### 4-1. 一括提案方式（全カテゴリ対応）
1. プロフィール設定で「仕事内容」を入力（必須）
2. ナビゲーションメニューから「目標作成」を選択
3. カテゴリを選択（挑戦目標・業務目標・人財目標）
4. 「AI提案を生成」ボタンをクリック
5. 生成された3-5件の提案を確認
6. 気に入った提案をクリックして選択
7. 右側のフォームに自動反映された内容を確認・編集
8. 「目標を保存」ボタンで保存（今後実装予定）

#### 4-2. 対話型アシスタント方式（挑戦目標・業務目標推奨）

**挑戦目標の場合:**
1. プロフィール設定で「仕事内容」を入力（必須）
2. ナビゲーションメニューから「目標作成」を選択
3. カテゴリで「挑戦目標」を選択
4. 「AI対話で挑戦目標を作成」ボタンをクリック
5. チャット形式で対話を開始
   - AIが3つの質問をします（挑戦領域・現状と目指す姿・成果の測定）
   - 質問に答えながら、段階的に目標を詳細化
   - 文字数調整やスケジュール詳細化などの要望に対応
6. 最終的なMBO文書をコピーしてフォームに貼り付け

**業務目標の場合:**
1. プロフィール設定で「仕事内容」を入力（必須）
2. ナビゲーションメニューから「目標作成」を選択
3. カテゴリで「業務目標」を選択
4. 「AI対話で業務目標を作成」ボタンをクリック
5. チャット形式で対話を開始
   - フェーズ1: 優先順位・組織目標・定量指標の3つの質問
   - フェーズ2: 目標範囲・役割・追加指標の3つの質問
   - フェーズ3: 算出方法・貢献の測り方・期限の確認（必要に応じて）
   - 不明点がすべて解消されたらMBO目標案を提示
6. 達成度評価基準や月次プロセスなどの追加支援を依頼可能
7. 最終的な目標文書をコピーしてフォームに貼り付け

**提案履歴の確認**
- ナビゲーションメニューから「提案履歴」を選択
- 過去のAI提案を確認・削除できます（一括提案方式のみ）

### 5. データのバックアップ
1. 設定画面を開く
2. 「データをエクスポート」ボタンをクリック
3. JSONファイルがダウンロードされます

### 6. データの復元
1. 設定画面を開く
2. 「データをインポート」ボタンをクリック
3. バックアップしたJSONファイルを選択
4. 確認後、データが復元されます

## 拡張性

このシステムは以下の拡張に対応しています：

1. **等級システムの拡張**
   - `src/types/grade.ts` で新しい等級を追加
   - `src/data/gradeMaster.ts` でマスタデータを更新

2. **新機能の追加**
   - `src/components/` に新しいコンポーネントを追加
   - `src/types/` に必要な型定義を追加

3. **データストレージの変更**
   - `src/utils/localStorage.ts` を変更することで、バックエンドAPIへの切り替えも可能

4. **AI機能のカスタマイズ**
   - `src/utils/openai.ts` のプロンプトを調整することで、分析内容をカスタマイズ可能
