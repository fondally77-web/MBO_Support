# MBO_Support

`MBO_Support` は、目標設定と週次振り返りを支援する自己管理アプリです。個人が「何を目指すか」「今週何を進めるか」「どう振り返るか」を一つの流れで扱えるようにし、行動が続く仕組みを作ることを目的としています。

## 状態

- 現在の位置づけ: active / 主軸
- 想定ユーザー: 学習、制作、仕事を自分で前に進めたい個人
- 方向性: 企業向け MBO システムではなく、まずは個人向け MVP から始める

## このリポジトリで目指すこと

- SMART や OKR の考え方を取り入れた目標設定
- 週次で進捗を振り返るレビュー導線
- 継続しやすいダッシュボードと自己管理体験
- 自己理解と実行管理をつなぐための基盤づくり

## 位置づけ

このリポジトリは、自己啓発・自己管理系の主軸です。

- `MBO_Support`: 目標を立てる、進捗を追う、振り返る
- `Cognitive_traits_test`: 自分の思考傾向や学び方を理解する

単体の診断で終わらせるのではなく、自己理解を実際の行動設計につなげるために、`MBO_Support` を中心に据えます。

## MVP で先に整える項目

1. 目標の作成、更新、完了管理
2. 週次レビューの入力と履歴確認
3. 進捗状況を見返せるシンプルなダッシュボード

## 次のタスク

1. 個人向け MVP の機能範囲を確定する
2. 週次振り返りの画面フローを定義する
3. `Cognitive_traits_test` の結果をどう参照するかを決める
